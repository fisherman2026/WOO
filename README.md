# World of Outdoorcraft (WOO)

  A mobile-first iOS RPG built with Expo for kids (ages 6-14) that gamifies exploring US National Parks.

  ## Features
  - 4 character classes: Ranger, Hiker, Naturalist, Tracker (male/female variants)
  - XP/leveling system
  - GPS, photo, lore, and explore quests
  - Inventory with rarity tiers
  - 2 parks: Yellowstone & Grand Teton
  - Multi-character profile system with per-profile progress saving
  - Complete 4-screen onboarding

  ## Built With
  - Expo / React Native
  - Expo Router
  - Reanimated
  - AsyncStorage

  ## Getting Started
  ```
  pnpm install
  pnpm --filter @workspace/world-of-outdoorcraft run dev
  ```
